Plugin Basics

新手先看：WooCommerce 和 Elementor Pro 是什么？

有些 Demo 会标注 WooCommerce 或 Elementor Pro，意思是落地网站时可能会用到对应插件能力。选 Demo 时不用先学技术，只要知道它们分别解决什么问题，以及是否会产生额外授权成本。

WooCommerce

WooCommerce 是 WordPress 上常用的电商插件，用来搭建商品、购物车、结账、订单等商城功能。适合跨境电商、产品在线购买、报价型商品目录等网站。

如果只是做企业展示站或询盘站，通常不一定需要它。

Elementor Pro

Elementor 是 WordPress 的可视化页面编辑器。Elementor Pro 是付费版本，部分模板会用到它的表单、弹窗、主题构建、动态内容等高级组件。

选到带 Pro 的 Demo 时，需要提前确认是否购买授权，或改成免费组件重做部分效果。

Licensed Toolkit

已购主题与插件，可按项目需要组合使用

这些工具主要用于后期真正落地 WordPress 网站：搭结构、做页面、扩展组件、管理产品字段、多语言和图片优化。具体是否启用，会根据客户选择的 Demo、网站类型和预算确认。

B

Blocksy Pro

常用的 WordPress 主题方案，适合用来搭建结构清晰、加载轻量、后期好维护的 B2B 独立站。它可以负责网站整体框架、页眉页脚、全局样式、响应式布局和基础性能优化。

主题基础
E

Elementor Pro

成熟的 WordPress 可视化页面编辑器，适合设计首页、产品页、落地页、询盘表单和弹窗等页面模块。Pro 版属于付费工具，使用前会根据项目需要确认授权和页面范围。

页面设计
UE

Unlimited Elements

为 Elementor 提供更多高级组件和内容模块，例如轮播、卡片、产品展示、图标模块、交互区块等。适合让 B2B 网站的产品卖点、应用场景和服务内容展示更丰富。

组件扩展
ACF

ACF Pro

适合管理产品参数、产品特点、应用场景、资料下载、FAQ、案例信息等结构化内容。对产品型号多、参数多、分类复杂的外贸 B2B 网站，可以提升后期维护效率。

内容字段
WPML

WPML

WordPress 多语言网站常用方案，适合需要英文、西班牙语、法语、德语等多语言版本的外贸网站。可以帮助管理不同语言的页面、菜单、产品内容和 SEO 信息。

多语言
SP

ShortPixel

常用的图片优化工具，可以压缩图片并生成 WebP / AVIF 格式，帮助网站减少图片体积、提升加载速度。对产品图很多的 B2B 网站，也有利于用户体验和 SEO 优化。
